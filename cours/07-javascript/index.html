Javascript

Qu'est-ce que c'est ?

  • Language de programmation principalement utilisé dans les pages web
  • Peut être utilisé dans d'autres domaines
  • Permet de d'écouter des événements et de modifier la page
  • Intégré à tous les navigateurs
  • Peut être désactivé
  • Diminutif : JS

Java is to Javascript what car is to carpet

Comment l'utiliser

Il existe plusieurs façon d'ajouter du javascript à une page

La plus simple étant de rajouter des balises <script></script> avant la fermeture du <body>

Il suffit ensuite d'écrire


<script>
    alert('Hello');
</script>

Comment ça marche

  • Il se lance dès qu'il est chargé
  • Si on recharge la page, il se lance à nouveau
  • Les espaces et retours à la ligne n'ont pas d'importance
  • Fonctionne avec des variables et des fonctions
  • Une mauvaise écriture du code peut empêcher l'intégralité du JS de fonctionner

Les variables

Javascript fonctionne avec des variables.
Elles permettent de sauvegarder des données

Ajoutez le code suivant entre les balises <script></script>


var toto = 'tata';
alert(toto);
                    

var toto = 'tata';
alert(toto);
  • var indique que l'on créé une variable
  • La variable toto contient la chaîne de caractères tata
  • Chaque instruction se termine par ;
  • alert() est une fonction permettant d'afficher le contenu dans une popup
  • Elle est déjà intégrée au JS
  • Une fonction s'appelle en rajoutant les ()
  • Ce qui se trouve entre ces () s'appellent les paramètres

alert('Salut');
alert('ça');
alert('va');
alert('?');

La technique du alert() n'est pas très pratique, nous allons utiliser la console

Tous les navigateurs récents intègrent une console

Elle a été conçue pour les développeurs et permet d'afficher le contenu des variables et d'écrire du JS directement dans la page

Pour la lancer :

  • Sur Chrome Mac : CMD + ALT + I
    Sur Chrome PC : F12
  • Aller dans l'onglet console
  • Pour afficher des données dans la console 
    
console.log('Salut');
console.log('ça');
console.log('va');
console.log('?');

Nomenclature

La nomenclature est l'art de nommer une variable

Sont autorisés :

  • Les lettres de A à Z en minuscule et majuscule
  • Les chiffres (sauf en premier caractère)
  • Les underscores _ (ou tirets du bas)

Bonnes pratiques

  • Nom clair et compréhensible
  • Pas trop long
  • En anglais

var user_name = 'Ben';
var user_age = 25;
var click_count = 10;
var image_width = 250;

Types de variables

Il existe plusieurs types de variables :

  • String
  • Number
  • Boolean
  • Array
  • Object
  • Function

String

  • Une String est une chaîne de caractères
  • On peut la délimiter par des ' ou des "
  • On peut les concaténer (rassembler) avec des +

var a = 'Bonjour';
var b = 'tout le monde';
var c = a + b;

console.log(c);

Number

  • Un Number est un simple nombre pouvant être un entier ou un nombre à virgule
  • On peut les additionner, multiplier, diviser, ...

var a = 2;
var b = 3.5;
var c = a + b;

console.log(c);

Boolean

  • Un Boolean est une valeur vraie ou fausse
  • On utilise true et false sans ' car ce n'est pas une String

var a = true;
var b = false;

console.log(a);
console.log(b);

Array

  • Un Array est un tableau de données
  • Il est délimité par [ et ]
  • Les valeurs sont séparées par des ,
  • Ces valeurs peuvent être de n'importe quel type (String, Number, Boolean et même un autre tableau)
  • On peut accéder à chacune de ces valeurs

var a = ['hey',true,25,['a','b']];

console.log(a);
console.log(a[0]);
console.log(a[1]);

Object

  • Un Object possède des propriétés qui peuvent contenir des données (???)
  • Il est délimité par { et }
  • Chaque propriété est définie par un nom et une valeur séparés par :
  • Les propriétés sont séparées par des ,
  • Si une propriété est une fonction, on appelle ça une méthode

var a = { color : 'red', distance : 10 };

console.log(a);
console.log(a.color);
console.log(a.distance);

Fonctions

  • Une fonction correspond à une série d'instructions
  • La syntaxe pour créer une fonction est différente d'une variable
  • Elle est déclarée avec function et les instructions sont entourées par { et }
  • Elle peut avoir des paramètres séparés par des ,
  • Elle peut renvoyer un résultat avec return

Fonctions


function addition(a,b)
{
    return a + b;
}

var c = addition(2,3);
console.log(c);

Il est possible de changer la valeur de toutes les variables que nous venons de voir


var a = 'bonjour';
console.log(a);

a = 10;
console.log(a);

a = false;
console.log(a);

Interval / Timeout

Parfois, on a besoin que javascript attende un peu avant d'éxecuter une instruction

La méthode setTimeout permet de faire ça

  • Le premier paramètre est une fonction, elle ne sera pas déclenchée tout de suite
  • Le deuxième paramètre est le temps à attendre avant de déclencher la fonction (en millisecondes)

setTimeout(function()
{
    console.log('Bonjour');
},1000);

Parfois, on a besoin que javascript déclenche une instruction plusieurs fois à interval régulier

La méthode setInterval permet de faire ça

Les paramètres sont les mêmes que pour le setTimeout


setInterval(function()
{
    console.log('Bonjour');
},1000);

If / ELSE

En javascript, il est possible de tester une condition et de faire telle action si la condition est vraie et telle autre action si la condition est fausse

Pour cela, on utilise if et else avec des accolades { }

Pour tester une égalité, il faut utiliser ==


var toto = 5;
if(toto == 5)
{
    console.log('vrai !');
}
else
{
    console.log('faux');
}

Si toto vaut 5, la console va afficher vrai !, sinon elle va afficher faux !

Pour tester une inégalité, il faut utiliser !=


var toto = 6;
if(toto != 5)
{
    console.log('coucou');
}

Si toto est différent de 5, la console va afficher coucou

Pour tester si une valeur est supérieure ou inférieure, il faut utiliser > et <


var toto = 6;
if(toto > 5)
{
    console.log('coucou');
}

Si toto est supérieur à 5, la console va afficher coucou

On peut tester autre chose que des Numbers


if(toto == false)
{
    console.log('coucou 1');
}

if(toto == 'test')
{
    console.log('coucou 2');
}

On peut comparer une variable à une autre


var toto = 5;
var tata = 10;
if(toto == tata)
{
    console.log('coucou');
}
